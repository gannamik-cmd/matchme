# Numerology Compatibility Bot

Telegram-бот для проверки совместимости имён и дат рождения с использованием нумерологии.

## Функционал
- Совместимость по именам
- Совместимость по датам рождения
- Подробный анализ и процент совместимости
- Поддержка русского и английского языков

## Установка на Render.com

### 1. Подготовка
1. Создайте бота через [@BotFather](https://t.me/botfather) в Telegram
2. Получите токен бота
